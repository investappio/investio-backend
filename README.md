# Invest.io Backend


## Getting Started

```
git clone https://github.com/Kushrati/investio-backend.git
cd investio-backend
echo "PORT=3000" > .env
echo "MONGO_HOSTNAME=mongo" > .env
echo "MONGO_DATABASE=..." > .env
echo "MONGO_USERNAME=..." > .env
echo "MONGO_PASSWORD=..." > .env
echo "REDIS_PASSWORD=..." > .env
echo "JWT_SECRET=..." > .env
echo "APCA_API_KEY_ID=..." > .env
echo "APCA_API_SECRET_KEY=..." > .env
docker-compose up -d
```
